# Rating

Microsserviço responsável pelo sistema de feedbacks para os usuários da aplicação.

## Colocando no ar

Com o Docker e Docker-Compose instalados, basta apenas utilizar os comandos:

1. Build
```shell
    make build
```
2. Executar
```shell
    make run
```
2.1 Executar em background
```shell
    make run-silent
```

## Rodando os testes

Com o Docker e Docker-Compose instalados, basta apenas utilizar os comandos:  

```shell
    make test
```
